What is it?

Infinipass is part password generator, part password manager. It can generate complex unique passwords based on something you know (your passphrase), and something about the service you are accessing (such as the domain name) and something unique to you (your username for the service).

How do I use it?

Create a book to generate multiple passwords with the same passphrase.

First, expand the left menu and click the + button.

Next, name the book and set your passphrase. The longer and more complex your passphrase the more difficult it would be for someone to crack it.

Any character is allowed, even foreign languages and emoji.

Finally, add each service you want to generate a password or PIN for.

What should I use for my passphrase?

Your passphrase should be something secret that you can remember. It can, and should, be long. It can have any combination of characters, including combinations of foreign languages and emoji.

Even though it is optional, it is highly recommended that you always include a username when generating a password. This can be the username that you use to access the service.

I just want to use it without saving anything

Use the Quick Generate tool.

What's with the emoji?

Is it safe?

The web version of this app does not send anything to any server. All of your information, including your passphrase and the list of websites and services you create, are only saved in your web browser.

It is very important to memorize your passphrase and not share it with anyone.

You may also opt to encrypt the local database from the Settings page. If the database is encrypted then a password must be provided each time the application is started.

The mobile app version also doesn't send anything to a server. The mobile app explicitly does not have permissions to access the Internet at all.

Since your information is only saved in your web browser or mobile device, if you need to access your passwords from another computer you will need to type your passphrase and add your services/websites again from that device.

How do I export or backup my database?

The database can be encrypted and exported for backup or for copying to another system from the Import-Export page.

How does it work?

  1. Your passphrase is combined with the name of the service and the username you use for the service, as well as the current version number, into a single string. Unless set otherwise, the service name and username are converted to lower case.
  2. The string is passed to WebCrypto's PBKDF2 function using SHA-512 and iterated 20480 times to generate a 128 bit hash value.
  3. The hash value is used to seed a random number generator (sfc32).
  4. The random number generator is used to generate each character of the password.
  5. Finally, the password shuffled.
